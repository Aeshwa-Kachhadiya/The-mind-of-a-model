# The Mind of a Model 🧠

A 10-day Machine Learning series documenting the complete journey from raw data to deployed model.

---

## The Series

- Day 01 → The ML Pipeline
- Day 02 → Data Understanding + Cleaning
- Day 03 → Feature Engineering
- Day 04 → Model Building
- Day 05 → Model Evaluation
- Day 06 → Overfitting + Underfitting
- Day 07 → Hyperparameter Tuning
- Day 08 → Model Comparison
- Day 09 → Model Deployment
- Day 10 → MLOps + What comes next

---

## Dataset

LinkedIn Post Performance Dataset

- 300 rows
- 20 features
- Goal: Predicting if a LinkedIn post will go viral
